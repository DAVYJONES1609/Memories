Player's Hub

Player's Hub is a web application where gamers can upload their gameplay videos, and viewers can watch and learn from them to improve their gameplay skills.

Installation-

To run the Player's Hub project locally, follow these steps:

    1-Clone the repository: git clone https://github.com/your-username/players-hub.gi

    2-Install dependencies for the frontend and backend: npm install--.

    3-Set up the database: Install MongoDB if you haven't already.

    4-Start the development server: npm run dev.

    5-Access the application at http://localhost:3000.

Usage-

Player's Hub provides the following functionalities:

    1-User authentication: Sign up, log in, and forgot password functionality.
    2-Video uploading: Gamers can upload their gameplay videos.
    3-Video viewing: Viewers can watch gameplay videos uploaded by gamers.
    4-Comments: Users can comment on videos to provide feedback or ask questions.
    5-Ratings: Users can rate videos on a scale of one to five stars.

For a visual demonstration of the project and its functionalities, watch the Project Demo Video.
